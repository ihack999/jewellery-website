Downloaded runtime textures for the custom jewellery design studio.

- `studio_small_08_1k.hdr` is the 1K HDR version of Poly Haven's Studio Small 08 HDRI, distributed as CC0.
- `Metal002/Metal002_1K-JPG_NormalGL.jpg` and `Metal002/Metal002_1K-JPG_Roughness.jpg` are runtime maps from ambientCG Metal 002, distributed as CC0.

Only the runtime maps used by the browser were kept.
